## 新特性

### C++11

#### 新关键字

1. `auto`自动类型推导
2. `decltype`从一个变量或表达式中得到类型，有点像`auto`的反函数
3. `nullptr`

#### 新特性

1. `range for`
   ```cpp
    for (TYPE element : COLLECTION)
    {  
        //.....
    }
   ```
2. 类型推导
3. Lambda表达式
4. 变长参数的模板
5. 字符串和数值的转换
6. 智能指针
7. 新增容器：`array`、`unordered_set`、`unordered_map`
8. `function`
