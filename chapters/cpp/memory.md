## 内存分配

### `new/delete`和`malloc/free`

| **特征** | **`new/delete`** | **`malloc/free`** |
| :---: | :---: | :---: |
| 分配内存的位置 | 自由存储区 | 堆区 |
| 正常返回值 | 完整类型指针 | `void*` |
| 失败返回值 | 默认抛出异常 | `NULL` |
| 分配内存的大小 | 由编译器根据类型计算得出 | 必须显式指定字节数 |
| 处理数组 | **内置** | 无 |
| 已分配内存的扩充 | 无法直观地处理 | 使用`realloc`简单完成 |
| 是否相互调用 | **可以** | 不可以 |
| 函数重载 | **可以** | 不可以 |
