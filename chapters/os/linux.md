## Linux常用命令

### 文件相关操作

1. `cd`：Change Directory
2. `pwd`：Print Working Directory
3. `mkdir`：Make Directory
4. `rmdir`：Remove **Empty** Directory
5. `ls`：List
6. `cp`：Copy
7. `rm`：Remove
8. `mv`：Move

### 内容查看

1. `cat`：Concatenate（连续）
2. `tac`
3. `nl`
4. `more`
5. `less`
6. `head`
7. `tail`
8. `touch`
9. `grep`

### 查看系统信息

1. `df`：Disk Free
2. `du`：Disk Usage
3. `ps`：Process Status
4. `top`：动态观察程序的变化

### 网络常用测试工具

1. `ping`
2. `tracert`
3. `telnet`
4. `netstat`
