## Java新特性

### Java 8

### 新特性

1. 引入Lambda表达式
2. `Metaspace`（`元空间`） 取代`PermGen`（`永久代`）
3. Stream API
4. 扩展方法
5. 并行数组

### 细节修改

1. HashMap使用散列表和红黑树实现。之前只使用了散列表。

### Java 9

### 新特性

1. Java平台级模块系统
2. `Garbage-First`（`G1`）垃圾收集器取代了`Parallel Collector`作为默认GC
3. 交互式编程语言`JShell`

### Java 10

### 新特性

1. 局部变量类型推断
2. 基于Java的`JIT`编译器
3. 并行全垃圾收集器`G1`
